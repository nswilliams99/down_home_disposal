## 1. Fix Page Load Speed

### A. Optimize Images

Check if any images are unoptimized. In any component using images, use Next.js Image component:

```tsx
import Image from 'next/image';

// Replace this:
<img src="/truck.jpg" alt="Trash truck" />

// With this:
<Image 
  src="/truck.jpg" 
  alt="Trash truck" 
  width={800} 
  height={600}
  priority={false}
  loading="lazy"
/>
```

For hero images (above the fold), use `priority={true}` instead.

### B. Lazy Load Third-Party Scripts

**File:** `app/layout.tsx`

If you have Google Analytics or other scripts, lazy load them:

```tsx
import Script from 'next/script';

// In your layout, after </body> or at end of component:
<Script
  src="https://www.googletagmanager.com/gtag/js?id=YOUR-GA-ID"
  strategy="lazyOnload"
/>
<Script id="google-analytics" strategy="lazyOnload">
  {`
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());
    gtag('config', 'YOUR-GA-ID');
  `}
</Script>
```

### C. Add Font Optimization

**File:** `app/layout.tsx`

Use Next.js font optimization:

```tsx
import { Inter } from 'next/font/google';

const inter = Inter({ 
  subsets: ['latin'],
  display: 'swap',
});

export default function RootLayout({ children }) {
  return (
    <html lang="en" className={inter.className}>
      {/* ... */}
    </html>
  );
}
```

### D. Check for Unused CSS/JS

Run in terminal:
```bash
npm run build
```

Look for large chunks. If any page bundle is over 100KB, investigate what's being imported.
